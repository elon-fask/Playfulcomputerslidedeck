## API Request/Response Flow

### Example: Fetching Presentations

**Request:**
```http
GET http://localhost:4000/api/presentations HTTP/1.1
Host: localhost:4000
Accept: application/json
```

**Backend Processing:**
```javascript
app.get('/api/presentations', (req, res) => {
  const rows = db.prepare('SELECT * FROM presentations ORDER BY created_at DESC').all();
  res.json(rows);
});
```

**Response:**
```json

  },
```

